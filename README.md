# Network Intrusion Detection System (NIDS)
### CodeAlpha Cybersecurity Internship — Task 4

A Python-based NIDS that simulates Snort/Suricata-style rule-based 
detection with real-time alerting and automated response mechanisms.

## Features
- 15 Snort-style detection rules
- Real-time packet inspection engine
- Auto-blocks CRITICAL threats instantly
- Attack types: SQL Injection, XSS, Command Injection, Metasploit, Directory Traversal, Botnet C2
- JSON alert logging

## How to Run
python nids_single.py

## Sample Output
- Packets analyzed: 200
- Alerts triggered: 10
- IPs auto-blocked: 5

## Tech Stack
- Python 3.x
- No external libraries required
